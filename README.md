## Como Usar

> Para clonar e rodar essa aplicação, você vai precisar do Git instalado em seu computador.

#### Clone o repositório

```
  git clone https://github.com/vleonecezar/art.git
```

#### Abra o repositório

```
  cd art
```

#### Rode a aplicação

```
  Abra o index.html em seu navegador
```
